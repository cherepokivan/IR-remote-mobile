## 🚀 Как использовать

### 1. Загрузите на GitHub
```bash
git add .
git commit -m "feat: implement all main screens and functionality"
git push origin main
```

### 2. GitHub Actions соберет APK автоматически
- Сборка запустится автоматически
- APK будет доступен в Artifacts
- Проверки пройдут автоматически

### 3. Скачайте APK из GitHub Actions
- Перейдите в Actions
- Выберите последний workflow run
- Скачайте `app-debug` из Artifacts

### 4. Установите на устройство с ИК-портом
```bash
adb install app-debug.apk
```

### 5. Запустите приложение
- При первом запуске загрузятся тестовые данные
- 3 устройства будут доступны сразу
- Можно сразу тестировать управление

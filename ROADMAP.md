## 📝 Известные проблемы

### Критические
- Нет

### Важные
- IR передатчик не определяется на некоторых устройствах (работа через reflection API)
- Загрузка IRDB ограничена 3 устройствами (нужно расширить)

### Незначительные
- Deprecated иконки (ArrowBack, ArrowForward) - нужно обновить на AutoMirrored версии
- Неиспользуемые переменные в IRDBMapper

- **ИК-порт**: Не все устройства имеют ИК-порт

### Митигация
- Проверка наличия ИК-порта при запуске
- Постепенная загрузка баз данных
- Унифицированный формат внутри приложения
- Эмулятор для UI тестов, реальное устройство для ИК

### Зависимости
- Android SDK 34
- Реальное устройство с ИК-портом
- Доступ к базам данных (GitHub)
- Время на разработку

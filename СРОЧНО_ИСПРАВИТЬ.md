# 🚨 СРОЧНОЕ ИСПРАВЛЕНИЕ ОШИБОК СБОРКИ

## Проблема
В репозитории есть старые файлы из пакетов `com.example.irremote` и `com.irremote.app`, которые вызывают ошибки компиляции.

## Решение (выберите один способ)

### Способ 1: Автоматический скрипт (Windows)
Просто запустите файл:
```
cleanup.bat
```

### Способ 2: Вручную через Git Bash
```bash
git rm -r --cached app/src/main/java/com/example/irremote
git rm -r --cached app/src/main/java/com/irremote/app
git add .
git commit -m "fix: remove duplicate packages and fix SettopBox icon"
git push
```

### Способ 3: Через PowerShell
```powershell
git rm -r --cached app/src/main/java/com/example/irremote
git rm -r --cached app/src/main/java/com/irremote/app
git add .
git commit -m "fix: remove duplicate packages and fix SettopBox icon"
git push
```

## После выполнения

GitHub Actions автоматически пересоберёт проект БЕЗ ОШИБОК! ✅

---

**Важно**: Нужно удалить старые файлы из git индекса, иначе они будут продолжать компилироваться на GitHub Actions.
